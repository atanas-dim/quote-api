## Features
* Fetch a Random Quote
* Fetch all Quotes
* Fetch by Author
* Add a New Quote
* Update a quote (via URLs and Postman)
* Delete a quote (via URLs and Postman)

## Start
Start server in node:
```
node server.js
```
then visit **localhost:4001** in your browser.
